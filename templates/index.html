What is this Lambton Chatbot?

Lambton Chatbot created as a Capstone Project. The idea behind this is to help a student to answer all their questions about the college coursework and other aspects.

Loved what we did?, This is How we did it...

RASA

We have used RASA Bot framework to create a artificial intelligent chatbot.

Web Development

We have used Python's Flask Web framework, HTML 5, CSS, JavaScript

Cloud Architecture

We have used Google Cloud Platform to run this Web and Chatbot Application
